GEOM preferences

In the MESH module you can set mesh preferences, default settings, which can be used in later sessions with this module.

 

 

Preferences for the Geometry module allow you to set the following parameters:

  • Default Shading Color  - follow the link to see the detailed information about the Select Color dialog box.

  • Step Value for Spin Boxes.

  • Default Display Mode - (wireframe or shading).
